GSAP Basics

This is a simple animation created using GSAP. The red circle will move horizontally, fade, and scale up and down in a smooth loop. You can customize the animation by changing the properties in the gsap.to() method in the JavaScript code. Try experimenting with different values for x, opacity, scale, duration, ease, delay, repeat, and yoyo to see how it affects the animation!

This animation uses the gsap.from() method, which animates the red circle from a specified state to its current state. The circle will start from being 300 pixels to the right, fully transparent, and scaled down to 0, and then it will animate to its original position, full opacity, and normal size. Like the previous animation, it will also loop indefinitely with a smooth back-and-forth motion. Feel free to modify the properties in the gsap.from() method to create your own unique animation!
